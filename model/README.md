# model

Inventor（モデルを作成するアプリケーション）などで作成したモデルをGAZEBOのシュミレーション内に反映させるための方法です。
<br>
これはwslではなくネイティブ環境のUbuntuで行っています。

---
## 動作環境と使用したアプリケーション

### Windows10
### Ubuntu 20.04.3
### Autodesk Inventor Professional 2021
### blender
---
## windows側での操作

1 Inventorで反映させたいモデルを作成する。（モデルの寸法の設定と作業平面に注意する）
2 作成したモデルをsdfファイルで保存する。
